# openlayer_lifemap
Display Lifemap using Bbox (ex T-rex) with Openlayers



## install Openlayers

`npm create ol-app my-app`

`cd my-app`

# copy files 

`cp ../index_lm.html index.html`

`cp ../main_lm.js main.js`

(for t-rex)

or 

`cp ../main_lm_bbox.js main.js`

(for bbox)

# start

`npm start`
